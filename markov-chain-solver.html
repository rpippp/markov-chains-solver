Quick guide

Editing: double-click the canvas to add a state; drag a state to move it; Shift + drag from one state to another creates an edge (to itself = a self-loop). The “Connect” button does the same with clicks only. Click an edge, or double-click its label, to edit its probability.

Navigation: scroll to zoom; drag the background to pan.

Solving: pick the start state and the X/Y sets — the result is recomputed instantly and exactly (linear system, Gaussian elimination with a residual check).

Computation modes

P(X before Y) — the probability that the chain, starting from the start state, reaches set X before set Y.

E[steps to X] — the expected number of transitions until X is first reached. If a closed class without X is reachable, the time is infinite and reported as such.

Distributions — stationary π (unique for irreducible chains), communicating-class classification, period, plus the step-by-step evolution of πₙ.

Simulation — an animated random walk through the chain, empirical visit frequencies compared against π, and a Monte Carlo estimator that cross-checks the exact result.
